Info

Works only on Apple pages.

An extension with bunch of tools. You can set the extension for your locale by right clicking the extension icon and selecting 'Options'.

Setting variables

  • locale_short_variable: This is what you have after www.apple.com/

    Example: www.apple.com/tr/ -> tr

  • page_locale_variable: Used for checking og:locale meta element.

    Example: tr_TR

  • page_lang_variable: Used for checking lang attributes.

    Example: tr-TR

  • sitename_variable: Used for checking site name meta element

    Example: Apple (KR)

support_link_variable: Used for checking support links

Example: ko-kr

https://support.apple.com/ko-kr/ipad

popup duration: Sets how long the passive popup stays on screen. 0 to turn off the popup.

Shortcuts

  • Command+shift+click: If you are on the US page, opens your locale's page. If you are on your locale's page, opens US page.
  • Command+option+click: Opens clicked elements background picture in a new tab.
